# Student Portfolio Images Directory

This directory contains placeholder images for the student portfolio website.

## Images Used:
- Professional headshot (index.html)
- Hobby-related images (hobbies.html)
- UMD campus and Duluth area photos (discover.html)
- Company office photos (career.html)

## Image Sources:
All images are sourced from Unsplash.com and used for educational/demo purposes.

## To Replace Images:
1. Replace placeholder URLs in HTML files with local image paths
2. Place actual images in this directory
3. Update image references in the HTML files accordingly

## Recommended Image Sizes:
- Profile images: 400x400px
- Gallery images: 300x200px
- Background images: 800x600px
